Class: Publication

A Publication represents basic information about a scientific article used in the Progenetix resource to indicate and annotate sources for genomic screening experiments.

URI: pgxpub:Publication

Attributes

Own

  • id 1..1
    • Range: String
  • label 0..1
    • Description: An informative text which e.g. can serve to indicate the meaning of id values but should not be used as identifier.
    • Range: String
  • ➞_id 0..1
    • Description: internal identifier; not part of the schema per se but included for convenience
    • Range: String
  • ➞pubmedid 0..1
    • Range: String
  • ➞pmcid 0..1
    • Range: String
  • ➞title 0..1
    • Range: String
  • ➞authors 0..1
    • Description: Author names, as usually represented in article citations (e.g. comma-concatenated).
    • Range: String
  • ➞journal 0..1
    • Description: Journal name, as usually represented in article citations.
    • Range: String
  • ➞pubYear 0..1
    • Range: String
  • ➞abstract 0..1
    • Range: String
  • ➞contact 0..1
    • Range: Contact
  • ➞status 0..1
    • Description: The status label is used to * disambiguate between Progenetix collection publications and "excluded" ones (i.e. not containing original genotyping data * add some information about reasons for the exclusion, e.g. for later access to currently out off scope articles
    • Range: String
  • ➞counts 0..1
    • Range: TechnologyCounts
  • ➞sample_types 0..*
    • Range: SampleTypeCount
  • ➞note 0..1
    • Range: String
  • ➞provenance 0..1
    • Range: String
  • ➞info 0..1
    • Range: String
  • ➞updated 0..1
    • Range: String
  • ➞progenetix_curator 0..1
    • Range: String
  • ➞progenetix_use 0..1
    • Range: String
